# Goodbase Flutter SDK

The official Goodbase client for Flutter and Dart applications. It provides
authenticated data access, durable per-user offline mutations, telemetry,
configuration, messaging, and app-attestation support.

## Requirements

- Dart 3.3 or newer
- Flutter 3.19 or newer
- A Goodbase application identifier and public API URL

Never embed administrator credentials, signing keys, or service-role secrets in
a client application. Use only application-scoped public configuration and
short-lived user sessions.

## Status

The package is source-ready and continuously compiled. Production publication
remains gated on Goodbase's real-device certification and signed release
process.
